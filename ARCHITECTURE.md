## Data Flow: Threat Detection to Response

### Step 1: Event Collection

```
Monitored System â†’ Wazuh Agent â†’ Wazuh Manager
```

- Agent collects logs (syslog, Windows events, application logs)
- Sends to Wazuh Manager for analysis

### Step 2: Alert Generation

```
Wazuh Manager â†’ Rule Engine â†’ Security Alert
```

- Applies detection rules
- Generates alerts for suspicious activity
- Maps to MITRE ATT&CK techniques

### Step 3: Alert Retrieval

```
SentinelX Backend â†’ Wazuh API â†’ Fetch Alerts
```

- Periodic polling or on-demand fetch
- Retrieves new security alerts

### Step 4: IOC Extraction

```
Alert â†’ Enrichment Service â†’ Extract IOCs
```

- Parses alert data
- Extracts IPs, domains, file hashes
- Identifies indicators of compromise

### Step 5: Threat Intelligence Enrichment

```
IOCs â†’ Threat Intel APIs â†’ Enriched Context
```

- **IP Addresses**:

  - AbuseIPDB â†’ Abuse confidence score
  - OTX â†’ Threat pulse count
  - VirusTotal â†’ Reputation score

- **Domains**:

  - VirusTotal â†’ Malicious verdict
  - OTX â†’ Associated threats

- **File Hashes**:
  - VirusTotal â†’ Antivirus detections

### Step 6: Threat Scoring

```
Enrichment Data â†’ Calculate Threat Score â†’ Risk Level
```

- Aggregates data from all sources
- Calculates composite threat score (0-100)
- Assigns risk level: Low, Medium, High, Critical

### Step 7: Data Storage

```
Enriched Alert â†’ MongoDB â†’ Persistent Storage
```

- Stores original alert
- Stores enrichment data
- Maintains historical context

### Step 8: Visualization

```
MongoDB â†’ API â†’ React Dashboard â†’ Security Analyst
```

- Real-time dashboard updates
- Threat trends and statistics
- Incident timelines

### Step 9: Notification

```
Critical Alert â†’ Slack Webhook â†’ SOC Team Notification
```

- Immediate notification for critical threats
- Includes threat score and recommendations

### Step 10: Response (Optional - Playbooks)

```
Enriched Alert â†’ Playbook Engine â†’ Automated Actions
```

- Block malicious IPs in firewall
- Quarantine infected files
- Isolate compromised systems
